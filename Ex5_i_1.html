We are used to assume that data follows the normal distribution, but there are domains where this is obviously incorrect, For instance, problems where variables have a maximum and a minimum value imposed by the nature of the data and where probabilites are skewed towards one of the extremes. In that case we can fit a more suitable distribution. For instance, the Gumbel distribution is used specifically for that kind of data. The probability distribution function of the Gumbel distribution is defined as:

$$ p\left(x\right) = \frac{1}{\beta}\ \exp\left(-\ \frac{(x-\mu)}{\beta}\right)\ \exp\left(-\ \exp\left(-\ \frac{(x-\mu)}{\beta}\right)\right) $$

def gumbel_pdf(x, mu, beta):
    # beta > 0
    return 1/beta*np.exp(-((x-mu)/beta))*np.exp(-(np.exp(-((x-mu)/beta))))

(b) Write the negative log likelihood of a sample and find the partial derivatives with respect to the parameters.

Sabem que cada element de la mostra segueix la distribuci de Gumbel, llavors la log-versemblana amb parmetres $ \theta = (\mu, \beta)$ s:

\begin{align*} \mathcal{L}(\theta) &= -\ \sum_{n=1}^N\ log\ \ Gumbel(x_n|\theta) = \newline &= -\ \sum_{n=1}^N\ log\ \frac{1}{\beta}\ \exp\left(-\ \frac{(x_n-\mu)}{\beta}\right)\ \exp\left(-\ \exp\left(-\ \frac{(x_n-\mu)}{\beta}\right)\right) = \newline &= -\ \sum_{n=1}^N\ log\ \frac{1}{\beta}\ \exp \left( -\ \frac{(x_n-\mu)}{\beta} -\ \exp\left(-\ \frac{(x_n-\mu)}{\beta}\right) \right)= \newline &= -\ \sum_{n=1}^N\ log\ \frac{1}{\beta}\ -\ \sum_{n=1}^N\ log\ \exp \left( -\ \frac{(x_n-\mu)}{\beta} -\ \exp\left(-\ \frac{(x_n-\mu)}{\beta}\right) \right) = \newline &= -\ \sum_{n=1}^N\ log\ \frac{1}{\beta}\ +\ \sum_{n=1}^N\ \left( \frac{(x_n-\mu)}{\beta} +\ \exp\left(-\ \frac{(x_n-\mu)}{\beta}\right) \right) = \newline &= -\ \sum_{n=1}^N\ ( log\ 1 - log\ \beta ) +\ \sum_{n=1}^N\ \left( \frac{(x_n-\mu)}{\beta} +\ \exp\left(-\ \frac{(x_n-\mu)}{\beta}\right) \right) = \newline &= \sum_{n=1}^N\ log\ \beta +\ \sum_{n=1}^N\ \left( \frac{(x_n-\mu)}{\beta} +\ \exp\left(-\ \frac{(x_n-\mu)}{\beta}\right) \right) = \newline &= N\ log\ \beta +\ \sum_{n=1}^N\ \left( \frac{(x_n-\mu)}{\beta} +\ \exp\left(-\ \frac{(x_n-\mu)}{\beta}\right) \right) = \newline &= N\ log\ \beta +\ \sum_{n=1}^N\ \frac{(x_n-\mu)}{\beta} -\ \sum_{n=1}^N\ \exp\left(\frac{(x_n-\mu)}{\beta}\right) = \newline &= N\ log\ \beta +\ \frac{1}{\beta} \sum_{n=1}^N\ x_n -\frac{N \mu}{\beta} -\ \sum_{n=1}^N\ \exp\left(\frac{(x_n-\mu)}{\beta}\right) \newline \end{align*}

Les derivades parcials sn:

(I) \begin{align*} \frac{\partial\mathcal{L}}{\partial\mu} &= \frac{\partial}{\partial\mu} \left( N\ log\ \beta +\ \frac{1}{\beta} \sum_{n=1}^N\ x_n -\frac{N \mu}{\beta} -\ \sum_{n=1}^N\ \exp\left(\frac{(x_n-\mu)}{\beta}\right) \right) = \newline &= - \frac{N}{\beta}\ - \frac{\partial}{\partial\mu}\left( \sum_{n=1}^N\ \exp\left(\frac{(x_n-\mu)}{\beta}\right) \right) \end{align*}

(II) \begin{align*} \frac{\partial\mathcal{L}}{\partial\beta} &= \frac{\partial}{\partial\beta} \left( N\ log\ \beta +\ \frac{1}{\beta} \sum_{n=1}^N\ x_n -\frac{N \mu}{\beta} -\ \sum_{n=1}^N\ \exp\left(\frac{(x_n-\mu)}{\beta}\right) \right) = \newline &= \frac{N}{\beta} -\ \frac{1}{\beta^2} \sum_{n=1}^N\ x_n +\ \frac{N\mu}{\beta^2} - \frac{\partial}{\partial\beta}\left( \sum_{n=1}^N\ \exp\left(\frac{(x_n-\mu)}{\beta}\right) \right) \end{align*}

(c) The partial derivatives result in a system of non linear equations that have to be solved numerically. For the same price we can fit directly the parameters of the distribution using numerical optimization. We are going to use the function that the scipy library provides for non linear function minimization (scipy.optimize.minimize) that optimizes a function that receives a vector of parameters, in this case we want to minimize the negative log likelihood that will be a function of the two parameters ($\mu,\beta$). The optimization method that you must use is Nelder-Mead. You will have to define a python function for the negative log likelihood that receives a vector with the parameters and the data sample.

We are going to apply it to the dataset California Housing that can be downloaded using the function fetch_california_housing from the scikit-learn library. If you plot the variables you will see that there are several variables that behave like the Gumbel distribution. def negative_loglikelihood(popt, X):
    # parameters
    mu   = popt[0]
    beta = popt[1]

    gumbel = 1/beta * np.exp(-(X*mu)/beta) * np.exp(-1*np.exp(-(X*mu)/beta))
    negL = -1 * np.sum(np.log(gumbel))
    return negL
